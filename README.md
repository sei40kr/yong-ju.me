# yong-ju.me

My portfolio website.

## Languages & Frameworks

- TypeScript
- [SolidStart](https://start.solidjs.com/getting-started/what-is-solidstart)
- [SolidJS](https://www.solidjs.com)

## Development

### Start Development Server

```sh
npm start
```

### Type Checking

```sh
npm run typecheck
```

### Linting

```sh
npm run lint
```

### Formatting

```sh
npm run format
```
